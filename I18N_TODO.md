# i18n(4개국어) 미적용 추적 — RetWork `public/index.html`

> 작성: 2026-06-16 (build 515 기준 조사 / 516에서 치리공개 모달 처리 시작)
> 목적: 일본어 기본값이라 **`data-i18n`/`t()` 누락 시 한·영·중 모드에서도 일본어(또는 하드코딩 한국어)가 그대로 노출**됨. 전수 목록 + 진행상황 추적.
> 규칙: **어드민(`adm*`/`admin*`/`apc*`/`ov-admin-*`)은 운영자용 한국어 고정 → 대상 아님(제외).**
> 줄번호는 조사 시점(build 515) 기준 **근사값** — 편집하며 이동함. 함수명/요소 id로 찾는 게 정확.
>
> 처리 방법:
> - 정적 HTML 요소 → `data-i18n="key"`(텍스트) / `data-i18n-placeholder="key"` / `data-i18n-title="key"`. 적용은 `applyLang()`(~14580). 키 중첩 자식 있으면 텍스트만 `<span data-i18n>`로 감싸기.
> - 동적 JS(innerHTML/textContent/toast/placeholder) → `t('key', {params})`. `{n}` 보간 지원.
> - 키는 `var I18N={ja,ko,en,zh}`(~13448)의 4블록 **모두**에 추가. 한 곳 빠지면 ja로 폴백됨.
> - 카테고리 라벨은 헬퍼 `_cpCatLabel(key)` (ct.cat.* 재사용, 선두 이모지 제거) 참고.
